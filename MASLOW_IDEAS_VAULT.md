## App Icon / Branding — Adaptive Identity
**Idea:** The Maslow app icon (and potentially the logo) adapts contextually.
- Pride month → rainbow version
- Language-specific → flag or locale color variant
- Seasonal → subtle palette shifts
- User-selectable avatar icon (person / superhero / goblin / whatever)
- Could extend to the suite experience itself — pick your "vibe" and the icon reflects it

**Why it's good:** Deeply on-brand for inclusivity. Maslow is "for everyone" — this makes that literal and visual.
**Why it's parked:** App icon changes require App Store review cycles every time a variant changes. The personalization layer needs the core UX to be solid first, and you need actual users to validate whether they want this. It's a good idea — build it after you open. Phase 2.

**Do NOT build before launch.**
